# Real Estate

A collection of tools for real estate data analysis, web scraping, pricing, etc.

## Scraping

Scraping tools for the following websites:
* Rentler - rental listings
* Redfin - on-market and previous sales
* KSL - Utah classified
* Craigslist
* Zillow
* Homie
